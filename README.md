

# 📊 Superstore Sales Performance Dashboard (Power BI)

## 📌 Overview

This project presents an interactive **Sales Performance Dashboard** built using **Power BI** based on the Superstore dataset.
The goal is to transform raw sales data into meaningful insights that support business decision-making.

---

## 🚀 Key Features

* 📈 **Sales Overview**

  * Total Sales: 2.30M
  * Total Orders: 9.994K
  * Total Profit: 286.40K
  * Profit Margin: 12.47%

* 🌍 **Regional Analysis**

  * Profit comparison across Central, East, South, and West regions
  * Identification of top-performing regions

* 🛍 **Category Analysis**

  * Sales distribution across:

    * Technology
    * Furniture
    * Office Supplies

* 📊 **Trend Analysis**

  * Year-wise sales growth visualization

* 🔍 **Sales vs Profit**

  * Relationship between revenue and profitability

* 🎯 **Interactive Filters**

  * Region filter
  * Category filter

---

## 💡 Key Insights

* 📈 West region is the top profit contributor
* ⚠️ Furniture category has high sales but lower profitability
* 📊 Sales show consistent growth from 2015 to 2017
